Supervisor

Supervisor APIs provide services to monitor and/or control apps. These Supervisor Services are usually used in conjunction with running sbhelper.

These APIs are available:

Service Description multi-app safe
App Control API control Legato apps
Application Information API Legato app info retrieval
Framework Control API control the Legato Framework
Linux Kernel Module API module load and unload
Warning
Beware of the security risks associated with granting an app access to these services. Apps that can control the state of other apps can seriously interfere with system operation. Sometimes, even knowledge about which apps are running can reveal a vulnerability.
