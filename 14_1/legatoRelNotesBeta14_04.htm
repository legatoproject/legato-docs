Legato 14.04 Features

Legato Beta release 14.04 added the following new features:

cgroups support
CPU and memory limits can be specified in .adef files. See memLimit and cpuShare.

Diagnostic tools
A new on-target tool for examining the memory usage of a process has been added. See Inspect Tool.

IPv6 data session
IPv6 data connections are now supported where available. See Modem Data Control API and Platform Adaptor API.

PLMN scan and selection
Scan and selection is now supported. See MRC platform adaptor.

WP7 voice call issues resolution
Audio voice calls can now be made on the WP7. See Audio Service

Audio API for IS2 interface
Audio for IS2 is now supported. See here

Config Tree API changes
The Config Tree API has been refactored, and applications using the old API will need some updates.
When upgrading from a previous version of Legato, you'll also need to remove your old database on the target - the file format has changed.

Language independent type definition
The interface generation tool now supports enhanced type definitions. See language independent ifgen.


Known Issues

Legato Beta release 14.04 has the following known issues:

  • IPv4 and IPv6 - can't be supported on a single RMNET data connection.
  • UART1 - is set to serial console and can't be re-tasked to an application serial port.
  • IPC - does not gracefully handle invalid sessions.
  • Link-time symbol - conflicts might result in runtime failures due to an incorrect variable or function.
  • CDMA SMS - is not supported.
  • SWIAPP - must be disabled for Legato audio feature to work. See Forum ADD LINK .
  • Legato framework supervisor - does not support real time processes.
  • Modem profile - the data connection modem setting profile has to be entered in Config DB before the data connection can be made from Legato (the default profile fails).
  • .adef - using the .adef files: section to designate that files use a different display file name inside the sandbox results in an extra directory level being introduced (e.g., "[r] stuff/foo /usr/share/bar" will result in a file "/usr/share/bar/foo" in the sandbox, but should be /usr/share/bar).
  • Resource limits - only CPU and memory limits can be configured; other resources can only use default values.

Yocto Updates

Yocto Linux Beta release 14.04 includes the following updates:

  • The Yocto build now uses opkg (instead of RMP) on-target - this saves significant flash space.
  • Generates version information into /etc/legato/version.
  • Rolls-up product-related fixes for WP7 and AR7.

For complete info, see the Legato documentation.
