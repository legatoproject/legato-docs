HTTP Get

This sample app demonstrates how to use HTTP Get to open a data connection and use libcurl.

It has a Legato adef file with executables, bindings, requires, and processes sections. The cdef contains sources and requires sections.

$ cd apps/sample/httpGet
$ make wp85
$ update httpGet.wp85.update <ip address>

If you're not using a Sierra Wireless WP85xx module, substitute wp85 with the your target identifier.
