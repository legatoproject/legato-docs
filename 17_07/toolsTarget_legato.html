legato

Use the legato tool to control the Legato Application Framework.

Usage

legato [start|stop|restart|status|version|help]

legato start

Starts the Legato Application Framework. 

legato stop

Stops the Legato Application Framework. 

legato restart

Restarts the Legato Application Framework. 

legato status

Displays the current running state (started, stopped), system state (good, bad, probation) and

system index of Legato.

legato version

Displays the current installed version. 

legato help

Displays usage help.
