Target Management Tools

Target Management tools are available to help manage the target: adding/removing apps, starting/stopping apps and updating firmware.

By installing your development environment with Leaf the environment variables will already be configured and profile aware. If you have set up your environment manually run bin/legs to add the host tools to the shell's executable path.
