    8Â  * In a POSIX environment, like Linux, file descriptors (fds) are used for most process I/O.
    9Â  * Many components need to be notified when one or more fds are

   15Â  * has methods to monitor fds reporting related events so they won't interfere with other software
   16Â  * sharing the same thread.

   19Â  *

   21Â  * A handler function and set of events is also provided to le_fdMonitor_Create().

   24Â     // Monitor for data available to read.

   99Â  * | POLLIN                       | Input is available from the socket                           |
