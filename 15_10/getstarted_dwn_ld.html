Download Files

You have to download the Legato app framework and Legato packages to install Legato from the command-line. The Sierra Wireless Source Legato page has useful links.

Framework

Download Legato application framework and sample apps (with open source code) from
legato.io.

Packages

Legato Packages are available at Legato Downloads.

The following packages are available:

  • Legato Linux source distribution
  • Linux PC i686 32-bit Toolchain files
  • Linux PC x86 64-bit Toolchain files
  • WP85 firmware packages
