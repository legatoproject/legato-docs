Synchronously open a session with a service. Does not wait for the session to become available if not available..

\hyperlink{le__messaging_8h_a2ee1410da1dc345c86d6958b6cdda5e1}{le\+\_\+msg\+\_\+\+Try\+Open\+Session\+Sync()} differs from \hyperlink{le__messaging_8h_a8c6480e3708e20e1f9da032a93c80bc0}{le\+\_\+msg\+\_\+\+Open\+Session\+Sync()} in that \hyperlink{le__messaging_8h_a2ee1410da1dc345c86d6958b6cdda5e1}{le\+\_\+msg\+\_\+\+Try\+Open\+Session\+Sync()} will not wait for a server session to become available if it is not already available at the time of the call. That is, if the client\textquotesingle{}s interface is not bound to any service, or if the service that it is bound to is not currently advertised by the server, then \hyperlink{le__messaging_8h_a2ee1410da1dc345c86d6958b6cdda5e1}{le\+\_\+msg\+\_\+\+Try\+Open\+Session\+Sync()} will return an error code, while \hyperlink{le__messaging_8h_a8c6480e3708e20e1f9da032a93c80bc0}{le\+\_\+msg\+\_\+\+Open\+Session\+Sync()} will wait until the binding is created or the server advertises the service (or both).

\begin{DoxyReturn}{Returns}

\begin{DoxyItemize}
\item L\+E\+\_\+\+O\+K if the session was successfully opened.
\item L\+E\+\_\+\+N\+O\+T\+\_\+\+F\+O\+U\+N\+D if the server is not currently offering the service to which the client is bound.
\item L\+E\+\_\+\+N\+O\+T\+\_\+\+P\+E\+R\+M\+I\+T\+T\+E\+D if the client interface is not bound to any service (doesn\textquotesingle{}t have a binding).
\item L\+E\+\_\+\+C\+O\+M\+M\+\_\+\+E\+R\+R\+O\+R if the Service Directory cannot be reached.
\end{DoxyItemize}
\end{DoxyReturn}
\begin{DoxyNote}{Note}
Only clients open sessions. Servers\textquotesingle{} must patiently wait for clients to open sessions with them.
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
If the client and server do not agree on the maximum message size for the protocol, a fatal error will be logged and the client process will be killed. 
\end{DoxyWarning}
