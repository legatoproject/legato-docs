Creates, opens and locks file.

If the file does not exist, it will be created with the file permissions specified in the arugment permissions (modified by the process\textquotesingle{}s umask). Refer to the P\+O\+S\+I\+X function open(2) for details of mode\+\_\+t\+:

\href{http://man7.org/linux/man-pages/man2/open.2.html}{\tt http\+://man7.\+org/linux/man-\/pages/man2/open.\+2.\+html}
