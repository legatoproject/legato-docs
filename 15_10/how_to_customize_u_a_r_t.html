Customize UART

The topic describes how to customize the WP710x Dev Kit UART port so it can be configured as an application serial port.

Syntax

The syntax is:

AT!MAPUART=a,b

Where

  • a is the service type
  • b is the UART number, either 1 or 2.

Here's a code sample setting 17 for the service type and 1 for the UART number:

AT!MAPUART=17,1

Reboot your target after running the command.

Run AT!MAPUART? to check the config settings.

Service Type

The current AT command spec defines the service as:

Service to map to UART:
0:  UART disabled
1:  AT Command service
2:  Diagnostic Message service
3:  Reserved
4:  NMEA service
5-15:  Reserved
16: Linux Console
17: Customer Linux application
